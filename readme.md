**Retro Meets Modern вАФ A Bring-Your-Own-ROM (BYOR) Browser Emulator**

GameArena is a sleek, client-side web application that allows you to play classic console games directly in your browser. Built on top of the powerful open-source **EmulatorJS** framework, it features a custom retro-cyberpunk aesthetic and runs entirely locally. Zero server uploads, zero latency, and maximum privacy.

## вЬ® Features

*   **Bring Your Own ROM (BYOR):** Uses standard Web APIs to load your local game files directly into the browser's memory. Your files are never uploaded to any server.
*   **Multi-System Support:** Play games from classic consoles including Game Boy Advance, NES, Super Nintendo, Sega Genesis, and Nintendo 64.
*   **Retro-Modern UI:** Features a custom dark-mode interface with neon accents, frosted glass panels, and a dynamic 16-bit Synthwave SVG splash screen.
*   **Plug & Play Controls:** Full keyboard support out of the box, plus an in-game menu to connect Bluetooth gamepads and remap controls.
*   **Save States:** Save and load your game progress directly within the browser's local storage.

## рЯЪА How to Run Locally

Because GameArena processes everything on the client side, getting it running is incredibly simple:

1. **Clone the repository:**
```bash
git clone https://github.com/DevAyu-Codes/GameArena.git
```
2. **Open the project:**
   Navigate to the project folder and open `index.html` in your web browser. (For the best experience, use an extension like VS Code's "Live Server").
3. **Play:**
   * Select your desired console system from the dropdown.
   * Click **Choose Local ROM File...** and select a legally backed-up ROM from your computer.
   * Click the screen and enjoy!
